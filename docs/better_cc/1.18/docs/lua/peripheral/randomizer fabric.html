randomizer fabric

A Randomizer wrapped as a peripheral allows for, basic interaction with adjacent randomizers.
This is the Fabric version.

getRandom()Source

Returns a random number that is also the redstone output of the Randomizer Block

Returns

  1. number The random number itself

Usage

  • Wrap a Randomizer and store its value in number

    local random = peripheral.find("randomizer") or error("No Randomizer attached", 0)
number = random.getRandom() -- This returns a random number between 1 and 15
toggle(state)Source

Sets the enabled state on the Randomizer Block

Parameters

  1. state boolean True or False

Usage

  • Wrap a Randomizer and turn it off

    local random = peripheral.find("randomizer") or error("No Randomizer attached", 0)
random.toggle(false)
